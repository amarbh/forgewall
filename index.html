About Forgewall

      
    

    

      
Forgewall turns a form into a FortiGate (FortiOS 7.x) CLI configuration script. It runs entirely in your browser — no backend, no account, no tracking.

      
Privacy

      
100% client-side. Forgewall makes no network requests at all, so you can save this page and run it completely offline. Your configuration is never written to disk or local storage — it lives only in the tab and disappears when you close it, unless you explicitly choose Export.

Before you deploy

      
The output is a starting point, not a finished configuration. Review every value, replace all <PLACEHOLDERS>, and test version-specific syntax (timezone index, profile names, options) on your own device in a lab before touching production.
